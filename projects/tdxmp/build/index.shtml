BOSS背景 >>

黑衣也好，铁甲也罢，离盟主守护的都是正义二字。信念之坚定，纵使两种截然不同的观点，在离盟主身上亦无矛盾。披坚执锐、抵御外寇、平息内祸，称得上侠之大者，为国为民。夜行衣下，却是以杀止杀，黑衣亦可行侠！

战术提示 >>

白昼和黑夜时的离玉堂将采用两种不同的战斗风格作战。白昼时攻击凶猛，需要及时规避，并且快速清理召唤的士兵。黑夜时则更需要战斗技巧，和被困的队友提前沟通好精锐刺客的击杀顺序，进行必要的伤害分担。

BOSS背景 >>

叶盟主身世坎坷，少时遭遇灭门之祸，其后又逢失爱之痛。纵是孤星照命，仍以剑破之。坚信大丈夫立于天地间，当以庇护天下为己任。责任所在，当无恶无畏，即使深渊万丈，以身饲魔，仍一往无前。

战术提示 >>

天下晦明，魔由心生。叶知秋的战斗中，既要对抗心魔侵蚀，以明月涤心；又要面对自然伟力，与海潮搏击。

BOSS背景 >>

天才少女曲无忆的朋友很少，能知晓其内心的人就更少了。无聊的江湖人暗中称其为绝不会笑、绝不能惹、绝不认输的“三绝仙子”。凭一身武功智计，独掌一盟，所思所求就是不让任何一个短暂的性命被牺牲。惯常以谋略解决问题，所以江湖中见过曲盟主心意双环绝学，特别是星相阵法的人极少。

战术提示 >>

曲无忆将星宿投影与阵法机关之术结合，自创星相阵法之术。引星力，切割时空，役使星相蜃影。战斗会被分割在星光内外分别进行，需要同时面对心意双环绝学和四象之力。

BOSS背景 >>

嘲天宫一战中，唐青枫遭受重创，被送到移花养伤。如今伤势痊愈，也赶来协助少侠解毒。依旧是潇洒不羁的少年，各位少侠别来无恙？

战术提示 >>

唐青枫深得唐门暗器机关武学精髓，又在傀儡合击之术上青出于蓝。他最多可驭使八枚随身傀儡，功用各不相同，结阵作战，如臂使指。和他战斗除了要躲避层出不穷的暗器机关外，一定要注意自己在傀儡阵中的方位。
